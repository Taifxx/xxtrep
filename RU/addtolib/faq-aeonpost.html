Проблема заключается в том, что в Aeon MQ6 постер берется исключительно из FanArt'а. В дополнении FanArt передается элементу listitem, который проигрывается функцией setResolvedUrl. в Kodi есть небольшой баг, относительно функции setResolvedUrl, который игнорирует инструкцию ListItem.setArt(...). Таким образом, заставить Aeon MQ6 видеть постеры можно только внеся коррективу в код самого скина:

Файл:  Kodi \ addons \ skin.aeonmq6 \ 720p \ IncludesVariables.xml

Блок:

     <variable name="value_thumbfullscreen">
     ...
     <value condition="!IsEmpty(Player.Art(thumb))">$INFO[Player.Art(thumb)]</value>
     </variable>


     (!)   Добавить строку нужно после других value condition, перед закрытием
             блока </variable>
