If you need, you can start ADD To Lib by key. For this use Keymap editor or other launcher.

Keymap editor example:

1. Launch Keymap editor add-on, select: Edit > Global > Add-ons > Launch: ADD To Lib
2. Select Edit key and press the desired key within five seconds
3. Press back three times and select Save

















Other launcher:

You can use any launcher which runs the add-on or can execute code:
RunAddon(context.addtolib)
