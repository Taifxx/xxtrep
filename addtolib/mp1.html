About:

Add-on to add links  to  the  TV shows and movies in Kodi video library from other add-ons.

Add-on creates links to episodes and movies as .strm files and places them in folders 'TV shows' and  'Movies'. To add you need  focuses on playable link  to movie file or to be in a folder containing the TV show episodes (season of TV show), open the context menu and use 'Add to Lib' menu item.











Add-on  creates  .strm  files  into  local  folders  'TV  Shows'  and  'Movies' (as default). Path: <home folder> \ userdata \ addon_data \ context.addtolib \ LIB \. You can change the folder names in the add-on settings.


   (!)   You must one-time add folders 'TV shows' and 'Movies' as sources in the Kodi video library, and choose
             scrapers for them.


Adding movie:

To add a movie, you need focuses on playable link to movie file, open the context menu and use 'Add to Lib' menu item. In add-on menu select 'Add to library as a movie'.  After action, the movie will be added to the 'Movies' folder (default).
