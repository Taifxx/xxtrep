Playback control:

In this add-on is realized the possibility of obtaining full information from the library for playable .strm files (posters, fan art, progress, etc.). For this feature when you start playback system launches ADD to Lib addon for playback control.

Sync:

Starting with version 1.0.12 ADD to Lib allows for sync video database and watched statuses with Dropbox. Thus, multiple remotely located devices unable to change video database and watched statuses without server.
