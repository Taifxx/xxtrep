Adding episodes (season):

To add the episodes  (TV show season), you need to be in a folder containing playable links to episodes files, open context menu of any of episodes and select 'Add to Lib'. In add-on menu select 'Add to library as TV show season'.

When add-on was able to determine name of the TV show and this TV show has exists, the episodes will be added to an existing TV show. Item 'Update episodes' is used to add new episodes from source to the already added episodes.

When you add episodes of new or exists TV show automatically added 'episodes source' (a link to the folder from which you add episodes). The add-on can scan this folder (source) and report about new episodes. One TV show may include a plurality of sources. The sources are managed in 'source management' menu item.

For the TV show manually can be added 'seasons source' (a link to folder containing a list of the seasons of the TV show). In this case, the add-on will scan folder for the presence of the new season.

TV show are managed in 'TV show management' menu item.


Update episodes (seasons) manually:

To check update of the current TV show, select 'Check for new episodes'. To check all TV shows you need using 'Global check for updates' menu item. If updates are detected, the add-on will display a list of them. If you select source of proposed update, the update will take place automatically.

You can disable check for new episodes or seasons in 'update management' menu item.


Update episodes (seasons) automatically:

To check update of the all TV shows in background mode (or during Kodi startup), you need turn-on this options in add-on settings. If add-on find new seasons in sources, add-on will show you appropriate message. Use 'Continue las update' to see it. 

On your choise you can set semiautomatic update mode disabling some update options in add-on settings.


Callable URL:

In this add-on is realized the possibility of obtaining full information from the library for playable .strm files (posters, fan art, progress, etc.). To use this feature is necessary to create links into .strm files that call this add-on when playing a file. In the case of playing .strm file with callable link during playback will be available to the data of the episode, TV show, a feature to store the position and 'Watched status'.
