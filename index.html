ADD To Lib

Context menu item to add links to TV shows and movies from other add-ons in Kodi library, as .strm files. The addition allows you to check for new episodes and seasons, manage added content, and to receive full information from the library to playable .strm files.

FAQ: Problem with posters on Aeon MQ6:

EN: The problem is that Aeon MQ6 is taken poster exclusively from FanArt. In add-on FanArt is transferring to the listitem element, which plays setResolvedUrl function. In Kodi there is a small bug, regarding setResolvedUrl function. Bug ignores ListItem.setArt(...) . Thus, make Aeon MQ6 to see the posters only by making adjustments in skin code.

Line for adding:

File: Kodi\addons\skin.aeonmq6\720p\IncludesVariables.xml

Block:

<variable name="value_thumbfullscreen">

...

<value condition="!IsEmpty(Player.Art(thumb))">$INFO[Player.Art(thumb)]</value>

</variable>

(!) Add it after other value condition, before closing the block </variable>

RU: Проблема заключается в том, что в Aeon MQ6 постер берется исключительно из FanArt'а. В дополнении FanArt передается элементу listitem, который проигрывается функцией setResolvedUrl. в Kodi есть небольшой баг, относительно функции setResolvedUrl, который игнорирует инструкцию ListItem.setArt(...). Таким образом, заставить Aeon MQ6 видеть постеры можно только внеся коррективу в код самого скина.

Строка для добавления:

Файл: Kodi\addons\skin.aeonmq6\720p\IncludesVariables.xml

Блок:

<variable name="value_thumbfullscreen">

...

<value condition="!IsEmpty(Player.Art(thumb))">$INFO[Player.Art(thumb)]</value>

</variable>

(!) Добавить ее нужно после других value condition, перед закрытием блока </variable>
