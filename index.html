Install Add-ons from XXTREP KODI Add-on Repository by Taifxx.

DOWNLOAD repository (only): xxtrep.repository

Available add-ons:

Context menu item to add links to TV shows and movies from other add-ons in Kodi library, as .strm files. The addition allows you to check for new episodes and seasons, manage added content, and to receive full information from the library to playable .strm files.
